# Earth Online v2.3.0 Android

这是触控优先的 Android 前端，不是把桌面窗口缩小后直接搬到手机。

## 已实现

- 主页、任务、08:00–24:00 半小时记录、恢复营地、更多世界五个底部页签
- 最低版 / 标准版 / 挑战版任务结算
- 现实产物、下一步动作、寻求帮助、高质量工作循环
- 生存日 / 正常日 / 冒险日
- 敌情报告与 90 秒恢复协议
- 角色属性、职业、技能、首领、人生地图、成就与七日统计
- SQLite 本地存档
- 导入桌面版 `earth.db`、导出手机存档

## 手机存档

应用使用 Android 私有应用目录保存 `earth.db`。可在“更多世界 → 存档与身份”中导入或导出。

## 在电脑预览手机版界面

```bash
python main.py
```

## 构建 APK

参见 `ANDROID_BUILD.md`。Qt 官方 Android 部署工具目前要求 Linux 或 macOS 主机。
